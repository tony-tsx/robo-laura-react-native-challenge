# Robo Laura - React Native Challenge

## Installation

install dependencies with yarn

```sh
yarn install
```

link anything you need with the command 
```
react-native link
```

## Run

- ### Android
  ```sh
  yarn android
  ```
  or
  ```sh
  npm run android
  ```

- ### IOS
  ```sh
  yarn ios
  ```
  or
  ```sh
  npm run ios
  ```
